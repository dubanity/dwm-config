[statuspadding](https://dwm.suckless.org/patches/statuspadding/)

Very minimal custom patch allowing for more precise resizing.

```
{ MODKEY|ShiftMask,             XK_h,      setmfact,       {.f = -0.01} },
{ MODKEY|ShiftMask,             XK_l,      setmfact,       {.f = +0.01} },
```
